# 기부 도서관 (SKU Library) 📚

## 주요 설명
- 데이터베이스를 활용한 도서관 시스템: MSSQL을 활용한 도서 관리, 대출/반납, 회원 관리가 가능한 종합 도서관 관리 시스템
- 해당 프로젝트는 미디어소프트웨어학과 기말고사 대체시험으로 8개의 팀 중 1등(만점)을 달성.
- **팀 구성**:
  - **팀장**: 김민석
  - **팀원**: 송채현, 이도현
- **작업 기간**: 2024년 11월 한 달
- **개발 환경**: Windows
